# 网约车平台司机评价信息管理系统 - 前端

基于 Vue 3 + Element Plus + Vite + ECharts 的前端项目。

## 启动

```bash
npm install
npm run dev
```

前端默认运行在 http://localhost:5173，自动代理 `/api` 到后端 `http://localhost:8088`。

## 技术栈

- Vue 3 + Composition API
- Element Plus 组件库
- ECharts + echarts-wordcloud 图表
- Vue Router 路由
- Axios HTTP 客户端
